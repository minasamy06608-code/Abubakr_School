# 🏫 نظام إدارة أرقام الجلوس المدرسية

نظام احترافي لإدارة أرقام جلوس الطلاب في المدارس الابتدائية والإعدادية.

## 🚀 تشغيل المشروع

```bash
npm install
npm run dev
```

ثم افتح المتصفح على `http://localhost:3000`

## 🔑 بيانات الدخول الافتراضية

- **اسم المستخدم:** `admin`
- **كلمة المرور:** `school2024`

يمكن تغييرها من صفحة الإعدادات بعد الدخول.

## 📋 الصفوف الدراسية المدعومة

| الكود | الصف |
|-------|------|
| g3p | الصف الثالث الابتدائي |
| g4p | الصف الرابع الابتدائي |
| g5p | الصف الخامس الابتدائي |
| g6p | الصف السادس الابتدائي |
| g1m | الصف الأول الإعدادي |
| g2m | الصف الثاني الإعدادي |

## 🌟 المميزات

- ✅ إضافة / تعديل / حذف الطلاب
- ✅ توليد أرقام جلوس تلقائي (أبجدياً)
- ✅ بحث وفلترة متقدمة
- ✅ استيراد من Excel
- ✅ تصدير إلى Excel
- ✅ تصدير PDF رسمي
- ✅ طباعة كشوف رسمية
- ✅ طباعة بطاقات QR
- ✅ لوحة تحكم بإحصائيات
- ✅ حفظ تلقائي محلي (localStorage)
- ✅ واجهة عربية RTL احترافية
- ✅ وضع داكن / فاتح
- ✅ تصميم متجاوب (موبايل + ديسكتوب)

## 🔧 Firebase (اختياري)

1. أنشئ مشروعاً على https://console.firebase.google.com
2. انسخ `.env.local.example` إلى `.env.local`
3. أضف بيانات Firebase في `.env.local`

> بدون Firebase: كل البيانات تُحفظ محلياً في المتصفح
